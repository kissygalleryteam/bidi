## changelog

* 2013-12-26 增加1.2版本，用于kissy 1.4.1，因为1.4.1改动过大，bidi没法再兼容下去
  现在使用的方案是，在bidi本地有一个1.4.0的xtemplate的本地模块
* 1.1 支持kissy1.4，因为1.4和1.3的Xtemplate的api有点不一致，所以，<em>1.1只能在1.4版本下使用</em>
反之亦然，1.0仅支持kissy 1.3.x
